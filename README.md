# Questoes_BR
Este repositório será referente ao jogo Questões-BR, um jogo de perguntas e respostas (quiz) online e acessível para deficientes visuais 
Neste jogo, os players poderão responder a diversas perguntas de diferentes categorias, podendo a cada jogada escolher a categoria desejada.
Nas próximas versões, poderemos ter também chat integrado para comunicação durante as partidas, ranking em tempo real, etc.
A cada partida, os jogadores também poderão criar um apelido temporário para usar no jogo, igual acontece no Kahoot.
